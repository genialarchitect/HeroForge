## How to Contribute

We welcome contributions from the security community! Please read our [Contributing Guidelines](CONTRIBUTING.md) before submitting pull requests.

### Ways to Contribute

- **Code Contributions**: New features, bug fixes, and improvements
- **Documentation**: Improve guides, fix typos, add examples
- **Bug Reports**: Report issues with clear reproduction steps
- **Feature Requests**: Suggest new capabilities for authorized security testing
- **Security Research**: Responsible disclosure of vulnerabilities

### Recognition

Contributors are recognized in:
- This file for ongoing participation
- Release notes for significant contributions
- Security advisories for responsible vulnerability disclosure
